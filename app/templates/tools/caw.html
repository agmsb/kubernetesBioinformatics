Cancer analysis workflow

Cancer analysis workflow (caw) is an open source pipeline used for detection of somatic variants from WGS data. It was developed by the National Genomics Infrastructure at SciLifeLab Stockholm, Sweden. The pipeline can begin the analysis either from raw FASTQ files, only from the realignment step, or directly with any subset of variant callers using recalibrated BAM files. Github repo
